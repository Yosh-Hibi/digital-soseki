# TEI Data and Related Files

This directory contains TEI-related data files and associated materials created or curated in projects involving Yoshitaka Hibi.
このフォルダには、日比嘉高が関わったTEI関係のデータファイル、関連ファイルを置いています。

The files in this directory are mainly intended for research use in Japanese literary studies and Digital Humanities. They may include TEI-encoded XML files, documentation, file lists, metadata, and other materials related to text encoding and textual analysis.

## Contents

- TEI-encoded XML files
- File lists and documentation
- Related metadata or auxiliary files
- Materials connected to TEI-based research projects

## Notes

- The structure and contents of this directory may be updated as the project develops.
- Detailed file information, when necessary, is provided in the README or file list placed in each subdirectory.
- For version history and recent changes, please refer to the GitHub commit history.
